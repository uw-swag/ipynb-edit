```python
    import numpy as np
    import pandas as pd
    import matplotlib.pyplot as plt

    from deepchecks.checks.methodology import *
    from deepchecks.base import Dataset

    # Example 1: SingleFeatureContribution
    df = pd.DataFrame(np.random.randn(100, 3), columns=['x1', 'x2', 'x3'])
    df['x4'] = df['x1'] * 0.5 + df['x2']
    df['label'] = df['x2'] + 0.1 * df['x1']
    df['x5'] = df['label'].apply(lambda x: 'v1' if x < 0 else 'v2')

    ds = Dataset(df, label_name='label')

    SingleFeatureContribution().run(ds)

    # Example 2: SingleFeatureContribution with custom parameters
    my_check = SingleFeatureContribution(ppscore_params={'sample': 10})
    my_check.run(dataset=ds)
    ```

    The above code snippets are examples of how to use the `SingleFeatureContribution` check in the `deepchecks` library. The first example uses a simple dataset, and the second example uses the `ppscore_params` parameter to specify custom parameters for the `SingleFeatureContribution` check.
