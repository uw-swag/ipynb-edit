It can be used to recommend properties to users based on their previous behavior, such as the most popular or highest-rated properties in a certain neighbourhood. The system can also analyze the price fluctuations in different months and the reviews scores of the properties in different neighbourhoods.

    Steps to successfully pull symlinks on Windows:
    1. Open File Explorer.
    2. Click on the folder where the symlink is located.
    3. Right-click on the symlink and select "Properties".
    4. In the Properties window, go to the "General" tab.
    5. In the "Link" field, enter the path to the actual file or directory.
    6. Click "OK" to save the changes.

    Please note that the above steps are for Windows. The process may vary slightly on other operating systems.
