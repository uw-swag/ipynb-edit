    ## Calculus Notebook

    ### 1.1 Understanding Differential Calculus

    #### 1.1.1 Derivatives

    ##### 1.1.1.1 Derivative of a Function

    The derivative of a function `f(x)` at a point `x` is the limit of the sequence of functions:

    `f'(x) = lim(h->0) [(f(x + h) - f(x)) / h]`

    where `h` is a small positive number.

    ##### 1.1.1.2 Derivative of a Power Function

    The derivative of a power function `f(x)` is:

    `f'(x) = lim(h->0) [(f(x + h)^p - f(x)^p) / h^p]`

    where `p` is a real number.

    ##### 1.1.1.3 Derivative of a Sine Function

    The derivative of a sine function `f(x)` is:

    `f'(x) = lim(h->0) [(f(x + h) - f(x)) / h]`

    #### 1.1.2 Integrals

    ##### 1.1.2.1 Integral of a Function

    The integral of a function `f(x)` from `a` to `b` is the limit of the sequence of functions:

    `∫(f(x)) dx = lim(t->0) [(f(x + t) - f(x)) / t]`

    where `t` is a small positive number.

    ##### 1.1.2.2 Integral of a Power Function

    The integral of a power function `f(x)` from `a` to `b` is:

    `∫(f(x)^p) dx = lim(t->0) [(f(x + t)^p - f(x)^p) / t^p]`
