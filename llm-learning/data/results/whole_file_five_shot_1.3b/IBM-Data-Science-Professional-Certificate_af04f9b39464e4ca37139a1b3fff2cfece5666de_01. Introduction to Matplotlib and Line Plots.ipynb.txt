```python
    # Importing necessary libraries
    import matplotlib.pyplot as plt

    # Creating a simple line plot
    x = [1, 2, 3, 4, 5]
    y = [1, 4, 9, 16, 25]

    plt.plot(x, y)
    plt.title('Simple Line Plot')
    plt.xlabel('X-axis')
    plt.ylabel('Y-axis')
    plt.show()
    ```
    This code will create a simple line plot with the x-axis representing the numbers 1 to 5 and the y-axis representing the squares of the numbers.
