    ## New Docs

    ### Overview

    The `pandas_profiling` package is a Python library that provides a high-level interface for generating detailed reports on pandas DataFrames. It is built on top of the popular `pandas` library.

    ### Installation

    You can install the `pandas_profiling` package using pip:

    ```python
    pip install pandas-profiling
    ```

    ## Usage

    The main function of the `pandas_profiling` package is the `profile_report` function. This function generates a detailed report on a pandas DataFrame.

    Here is a simple example:

    ```python
    import pandas_profiling

    df = pandas.read_csv('your_file.csv')
    report = pandas_profiling.profile_report(df)
    report.to_file('report.html')
    ```

    This will generate a report in HTML format.
