    ## README
    This repository contains examples of using the `evidently` library to analyze and visualize the drift in the Iris dataset.

    ## Examples
    The following examples show how to use the `evidently` library to analyze and visualize the drift in the Iris dataset.

    - `iris_data_drift_report.show()`: This will display a report of the data drift in the Iris dataset.
    - `iris_data_and_target_drift_report.show()`: This will display a report of the data and target drift in the Iris dataset.

    The reports are saved as HTML files in the `reports` directory.

    ## Installation
    To install the `evidently` library, use the following command:
    ```
    pip install evidently
    ```
