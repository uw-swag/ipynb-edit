    ## Introduction

    This is a documentation for a Python package that provides a simple way to perform a Collatz operation.

    ## Installation

    You can install the package using pip:

    ```python
    pip install collatz
    ```

    ## Usage

    The `collatz` function can be used to perform a Collatz operation on a number. Here is an example:

    ```python
    import collatz

    result = collatz.collatz(10)
    print(result)  # Output: 5
    ```
