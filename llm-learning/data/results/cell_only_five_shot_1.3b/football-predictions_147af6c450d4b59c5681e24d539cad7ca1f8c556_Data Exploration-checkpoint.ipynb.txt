
    ## README
    
    ## Project Overview
    This project is a part of the Udacity Data Scientist Nanodegree program. The goal of this project is to predict the outcome of a soccer match based on various features. The data used in this project is from the 2018-2019 season.

    ## Data
    The data is divided into two parts:

    1. The 'season1819.csv' file contains the data from the 2018-2019 season.
    2. The 'season1718.csv' file contains the data from the 2017-2018 season.

    ## Installation
    The project requires Python 3.6 or above, and the following Python libraries: pandas, numpy, matplotlib, sklearn, and getml.

    ## Usage
    The project can be run using a Python environment. The main script is 'soccer_prediction.py', which can be run using the command:
    ```
    python soccer_prediction.py
    ```
