```
    ## Problem Statement
    # In this problem, you are required to implement a recommendation system using collaborative filtering. The system should be able to recommend items to a user based on their past behavior.

    ## Objective
    # The objective of this problem is to predict the rating a user would give to an item.

    ## Input
    # The input to the system is a matrix of user-item interactions. Each row represents a user's past behavior and each column represents an item. The value in the cell represents the user's rating given to the item.

    ## Output
    # The output of the system is a matrix of predicted ratings. Each cell represents the predicted rating a user would give to an item.

    ## Constraints
    # The system should be able to handle a large number of users and items.

    ## Assignment
    # The assignment is due on the 10th of July, 2022.
    ```
