```python
    import matplotlib.pyplot as plt

    # Create a simple line plot
    x = [1, 2, 3, 4, 5]
    y = [1, 4, 9, 16, 25]

    plt.plot(x, y)
    plt.show()
    ```
    In the above code, we are using matplotlib's `plt.plot()` function to create a simple line plot. The `x` and `y` variables are lists that represent the x and y coordinates of the points on the line. The `plt.show()` function is used to display the plot.
