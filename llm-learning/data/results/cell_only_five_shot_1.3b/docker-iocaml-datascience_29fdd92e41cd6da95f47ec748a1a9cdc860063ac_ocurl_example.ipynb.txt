```
    #!/bin/bash
    # ocurl is a command-line tool for making HTTP requests.
    # It's a part of the 'oc' package, which is a collection of tools for working with Open Container Initiative (formerly Docker) images.
    # For more information, please visit: https://github.com/containers/oc

    # Install ocurl
    curl -L https://raw.githubusercontent.com/containers/oc/master/hack/install.sh | sh

    # Verify the installation
    ocurl version
    ```
    This script will install the 'oc' command-line tool and then verify its installation.

    Please note that this script should be run in a Linux environment.

    Also, please be aware that running scripts from the internet is generally not recommended due to security and privacy concerns.
